# Выполненные задачи в рамках курса Web разработки от компании YADRO
## Для каждой задачи созданна отдельная ветка
### [Task-1]
Создать страницу с текстом, где подчеркивание слов реализовано не стандартным способом (с помощью text-decoration: underline;), а с помощью кастомных CSS-стилей. Подчеркивание должно имитировать стандартное, но создаваться с помощью других CSS-свойств.
